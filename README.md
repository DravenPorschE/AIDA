## Android Studio Git Setup

## Clone the project
```bash
git clone https://github.com/<your-username>/<repo-name>.git
````

## Update before coding

```bash
git pull origin main
```

## Save changes

```bash
git add .
git commit -m "your message"
```

## Push changes

```bash
git push origin main
```
